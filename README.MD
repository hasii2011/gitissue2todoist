gitissue2todoist
================


Converts Git issues to Todoist tasks

This OS X and IOS application can read your GitHub repositories and convert your GitHub issues to Todoist tasks. See the [wiki](https://github.com/hasii2011/PyGitIssue2Todoist/wiki/How-to-use-PyGitIssue2Todoist) for additional documentation.

IDE Setup
---------
When setting up this project in an IDE (like PyCharm or VSCode), you should install the dependencies into your local virtual environment so that the IDE can correctly resolve imports (like ``toga``) and provide autocomplete.

Run the following command in your active virtual environment:

.. code-block:: bash

    pip install -r requirements-dev.txt
